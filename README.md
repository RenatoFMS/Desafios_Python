#  Desafios de Python

Repositório destinado ao armazenamento de desafios de código realizados utilizando a linguagem **Python**.

##  Organização
O repositório é dividido por **temas principais**. Dentro de cada tema, os desafios são organizados em três níveis de complexidade:
* **Básico** 
* **Intermediário** 
* **Avançado**
